---
title: "Get Product Preview List"
openapi: "GET /v2/product_previews"
---

Retrieve paginated list of product previews with filtering and sorting options. Previews are generated videos that help you choose the best style before final rendering.

## Example Usage

```bash
curl --request GET \
  --url 'https://api.jogg.ai/v2/product_previews?page=1&page_size=10&status=4' \
  --header 'x-api-key: YOUR_API_KEY'
```
