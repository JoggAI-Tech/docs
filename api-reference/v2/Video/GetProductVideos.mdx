---
title: "Get Product Video List"
openapi: "GET /v2/product_videos"
---

Retrieve paginated list of generated product videos with filtering and sorting options.

## Example Usage

```bash
curl --request GET \
  --url 'https://api.jogg.ai/v2/product_videos?page=1&page_size=10&status=4' \
  --header 'x-api-key: YOUR_API_KEY'
```
