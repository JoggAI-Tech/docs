---
title: "List Template Videos"
openapi: "GET /v2/template_videos"
---

Get paginated list of template videos with optional filtering and sorting.

## Example Usage

```bash
curl --request GET \
  --url 'https://api.jogg.ai/v2/template_videos?page=1&page_size=10&status=4' \
  --header 'x-api-key: YOUR_API_KEY'
```
