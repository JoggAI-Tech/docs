---
title: "Get Custom Avatars"
openapi: "GET /open/v2/avatars/custom"
---

Retrieve the list of your custom avatars including photo avatars and product avatars.

## Example Usage

```bash
curl --request GET \
  --url 'https://api.jogg.ai/open/v2/avatars/custom' \
  --header 'x-api-key: YOUR_API_KEY'
```
