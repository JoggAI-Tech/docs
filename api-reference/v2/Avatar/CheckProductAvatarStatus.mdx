---
title: "Check Product Avatar Status"
openapi: "GET /v2/product_avatar"
---

Get the status and details of image to product avatar conversion.

## Example Usage

```bash
curl --request GET \
  --url 'https://api.jogg.ai/v2/product_avatar?motion_id=motion_123456' \
  --header 'x-api-key: YOUR_API_KEY'
```
