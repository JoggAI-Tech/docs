---
title: "Get AI Script Result"
openapi: "GET /open/v2/ai_scripts/results/{task_id}"
---

Retrieve the generated AI scripts by task ID.

## Example Usage

```bash
curl --request GET \
  --url 'https://api.jogg.ai/open/v2/ai_scripts/results/task_abc123' \
  --header 'x-api-key: YOUR_API_KEY'
```
