## Overview

Create personalized avatar videos using custom photo avatars generated from your own portrait photos. Perfect for branded content, personal messaging, or using specific talent.

<Info>
**Prerequisites:** You must first create a photo avatar before using it in videos.
</Info>

---

## Prerequisites

Before creating videos with photo avatars:

1. ✅ Create a photo avatar first
2. ✅ Get the photo avatar `avatar_id`
3. ✅ Prepare your script

---

## Quick Example

### Step 1: Get Your Photo Avatar ID

List your photo avatars:

```bash
curl --request GET \
  --url 'https://api.jogg.ai/open/v2/avatars/photo_avatars' \
  --header 'x-api-key: YOUR_API_KEY'
```

**Response:**

```json
{
  "code": 0,
  "msg": "Success",
  "data": {
    "avatars": [
      {
        "avatar_id": 123,
        "name": "My Custom Avatar",
        "status": "completed",
        "cover_url": "https://res.jogg.ai/avatars/cover_123.jpg"
      }
    ]
  }
}
```

---

### Step 2: Create Video with Photo Avatar

Use the photo avatar in video creation:

```bash
curl --request POST \
  --url 'https://api.jogg.ai/open/v2/create_video_from_avatar' \
  --header 'x-api-key: YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "avatar_id": 123,
    "avatar_type": 1,
    "voice_id": "en-US-ChristopherNeural",
    "script": "Hello! This is my custom photo avatar speaking.",
    "aspect_ratio": "portrait",
    "screen_style": 1,
    "caption": true
  }'
```

**Response:**

```json
{
  "code": 0,
  "msg": "Success",
  "data": {
    "video_id": "video_123456"
  }
}
```

<Check>
Set `avatar_type: 1` to indicate this is a custom avatar (photo avatar)
</Check>

---

## Key Parameters

| Parameter | Value | Description |
|-----------|-------|-------------|
| `avatar_id` | integer | Your photo avatar ID |
| `avatar_type` | `1` | **Important:** Must be `1` for custom/photo avatars |
| `voice_id` | string | Voice ID for text-to-speech |
| `script` | string | Text for avatar to speak |
| `aspect_ratio` | string | portrait/landscape/square |
| `screen_style` | integer | 1=Background, 2=Green screen, 3=WebM |

## Comparison: Public vs Photo Avatars

| Feature | Public Avatars | Photo Avatars |
|---------|---------------|---------------|
| **Creation** | Ready to use | Create from photo first |
| **Customization** | Pre-defined | Your own face |
| **avatar_type** | `0` | `1` |
| **Use Case** | Quick, generic | Personal, branded |
| **Cost** | Standard | May cost more credits |

## Tips for Best Results

<Tip>
**Photo Avatar Quality:**
- Use high-quality source photos
- Ensure photo avatar generation completed successfully
- Test with short script first
- Check avatar list for `status: "completed"`
</Tip>

**Script Recommendations:**
- Keep scripts natural and conversational
- Avoid very long scripts (< 2 minutes)
- Test pronunciation of special terms
- Use appropriate voice for avatar personality

## Troubleshooting

<AccordionGroup>
  <Accordion title="Avatar not found">
    **Error**: `Avatar ID does not exist`
    
    **Solution:**
    - List photo avatars to get correct ID
    - Ensure `avatar_type: 1` is set
    - Check avatar status is "completed"
    - Verify avatar belongs to your account
  </Accordion>
  
  <Accordion title="Avatar still processing">
    **Error**: `Avatar not ready`
    
    **Solution:**
    - Photo avatar generation can take 2-5 minutes
    - Check status using Get Photo Avatars endpoint
    - Wait for status to be "completed"
    - Don't create videos with "processing" avatars
  </Accordion>
  
  <Accordion title="Quality issues">
    **Issue:** Video quality doesn't look good
    
    **Solution:**
    - Recreate photo avatar with higher quality photo
    - Ensure source photo meets quality requirements
    - Use good lighting in source photo
    - See [Photo Avatar guide](/api-reference/v2/API%20Documentation/CreatePhotoAvatar) for photo tips
  </Accordion>
</AccordionGroup>
