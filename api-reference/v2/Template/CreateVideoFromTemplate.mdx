## Quick Start

### Step 1: Browse Available Templates

```bash
curl -X GET 'https://api.jogg.ai/v2/templates/custom' \
  -H 'x-api-key: YOUR_API_KEY'
```

### Step 2: Get Template Details

```bash
curl -X GET 'https://api.jogg.ai/v2/template/custom/tmpl_123' \
  -H 'x-api-key: YOUR_API_KEY'
```

### Step 3: Create Video

```bash
curl -X POST 'https://api.jogg.ai/v2/create_video_with_template' \
  -H 'x-api-key: YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "template_id": 1234,
    "video_name": "Product Showcase",
    "avatar_id": 127,
    "avatar_type": 0,
    "voice_language": "english",
    "voice_id": "en-US-ChristopherNeural",
    "variables": [
      {
        "type": "text",
        "name": "product_name",
        "properties": {
          "content": "Amazing Widget"
        }
      },
      {
        "type": "image",
        "name": "product_image",
        "properties": {
          "url": "https://example.com/widget.jpg"
        }
      },
      {
        "type": "script",
        "name": "script",
        "properties": {
          "content": "Check out our amazing product!"
        }
      }
    ]
  }'
```

**Response:**

```json
{
  "code": 0,
  "data": {
    "video_id": "vid_abc123"
  }
}
```

### Step 4: Get Video Status

```bash
curl -X GET 'https://api.jogg.ai/v2/template_video/vid_abc123' \
  -H 'x-api-key: YOUR_API_KEY'
```

## Request Parameters

### Required

| Parameter | Type | Description |
|-----------|------|-------------|
| `template_id` | integer | Template ID from library |
| `voice_language` | string | Voice language for text-to-speech |
| `variables` | array | Template variables array (see Variable Properties below) |

### Optional

| Parameter | Type | Description |
|-----------|------|-------------|
| `video_name` | string | Name for your video |
| `avatar_id` | integer | Avatar ID for the video |
| `avatar_type` | integer | 0: Public avatar, 1: Custom avatar |
| `voice_id` | string | Voice ID for narration |
| `captions_enabled` | boolean | Enable subtitles |

### Variable Properties

| Property | Type | Description |
|----------|------|-------------|
| `content` | string | For text/script type: Required, the text content. For image/video type: Optional, can be used as URL |
| `url` | string | Media resource URL for image/video type (choose url or asset_id) |
| `asset_id` | integer | Asset ID for image/video type, obtained from /v2/upload/asset (choose url or asset_id) |
