### Test Your API Key
Include your API key in the request header:

```bash
curl -X GET 'https://api.jogg.ai/open/v2/user/whoami' \
  -H 'x-api-key: YOUR_API_KEY'
```

**Expected Response:**

```json
{
  "code": 0,
  "msg": "Success",
  "data": {
    "userId": "user_123",
    "email": "user@example.com",
    "credits": 100
  }
}
```

## Quick Start Example

Here's a simple example to create an avatar video:

```bash
curl -X POST https://api.jogg.ai/api/v2/video/avatar \
  -H "x-api-key: your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "avatar_id": "avatar_123",
    "script": "Hello! Welcome to JoggAI API.",
    "voice_id": "voice_456"
  }'
```
