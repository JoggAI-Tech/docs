## Response Structure

All API responses follow a consistent structure:

```json
{
  "code": 0,
  "msg": "Success",
  "data": {...}
}
```

- `code: 0` = Success
- `code > 0` = Error

---

## Error Code Categories

| Category | Code Range | Description |
|----------|------------|-------------|
| **Authentication** | 101xx | API key issues |
| **Request Format** | 102xx | Invalid parameters or JSON |
| **Resources** | 103xx | Resource not found or limits |
| **Rate Limiting** | 104xx | Too many requests |
| **Credits** | 105xx | Insufficient credits |
| **Server Errors** | 106xx | Internal server issues |

## Common Error Codes

### Authentication Errors (101xx)

| Code | Message | Cause | Solution |
|------|---------|-------|----------|
| 10104 | API key is required | No `x-api-key` header | Add header: `x-api-key: YOUR_API_KEY` |
| 10105 | Invalid API key | Wrong or malformed key | Verify API key in Dashboard |
| 10106 | API key has expired | Key revoked or expired | Generate new API key |
| 10107 | Unauthorized access | Insufficient permissions | Check subscription plan |

### Request Errors (102xx)

| Code | Message | Cause | Solution |
|------|---------|-------|----------|
| 10201 | Invalid request format | Malformed JSON | Check JSON syntax and Content-Type header |
| 10202 | Missing required parameter | Required field not provided | Review API documentation for required fields |
| 10203 | Invalid parameter value | Value doesn't meet requirements | Check parameter type and constraints |

### Resource Errors (103xx)

| Code | Message | Cause | Solution |
|------|---------|-------|----------|
| 10301 | Resource not found | Invalid resource ID | Verify resource ID is correct |
| 10302 | Resource already exists | Duplicate resource | Use different identifier or update existing |
| 10303 | Resource limit exceeded | Maximum resources reached | Delete unused resources or upgrade plan |

### Rate Limit Errors (104xx)

| Code | Message | Cause | Solution |
|------|---------|-------|----------|
| 10401 | Rate limit exceeded | Too many requests | Wait and retry with exponential backoff |
| 10402 | Too many concurrent requests | Exceeded simultaneous limit | Reduce concurrent requests |

### Credit Errors (105xx)

| Code | Message | Cause | Solution |
|------|---------|-------|----------|
| 10501 | Insufficient credits | Not enough credits | Purchase additional credits |
| 10502 | Monthly credit limit exceeded | Used all monthly credits | Wait for next billing cycle or upgrade |

### Server Errors (106xx)

| Code | Message | Cause | Solution |
|------|---------|-------|----------|
| 10601 | Internal server error | Unexpected server error | Retry after short delay, contact support if persists |
| 10602 | Service temporarily unavailable | Maintenance or issues | Check status page, retry later |

## Getting Help

If you encounter an error:

1. Check the error message details
2. Review your request parameters
3. Consult the API Reference
4. Contact [support@jogg.ai](mailto:support@jogg.ai) with:
   - Error code and message
   - Request ID (if provided)
   - Timestamp
   - API endpoint

<Note>
Include your user ID (but not your API key) when contacting support.
</Note>
