---
title: "Get Product Avatars"
openapi: "GET /v2/avatars/product_avatars"
---

Get list of user's product avatars with optional aspect ratio filter.

## Example Usage

```bash
curl --request GET \
  --url 'https://api.jogg.ai/v2/avatars/product_avatars?aspect_ratio=portrait' \
  --header 'x-api-key: YOUR_API_KEY'
```
