---
title: "List Template Videos"
openapi: "GET /v2/template_videos"
---

Get paginated list of template videos with optional filtering and sorting.

## Example Usage

```bash
curl --request GET \
  --url 'https://api.jogg.ai/v2/template_videos?page=1&page_size=10&status=4' \
  --header 'x-api-key: YOUR_API_KEY'
```

**Response:**


## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `page` | integer | ❌ | Page number (default: 1) |
| `page_size` | integer | ❌ | Items per page, 1-100 (default: 10) |
| `status` | integer | ❌ | Filter by status: 3=Processing, 4=Success, 5=Failed |
| `sort_by` | string | ❌ | Sort field (default: "created_at") |
| `sort_order` | string | ❌ | Sort order: "asc" or "desc" (default: "desc") |
