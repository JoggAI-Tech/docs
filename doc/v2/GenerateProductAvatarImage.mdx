---
title: "Generate Product Avatar Image"
openapi: "POST /v2/product_avatar/generation"
---

Generate product avatar images using AI with specified parameters.

## Example Usage

```bash
curl --request POST \
  --url 'https://api.jogg.ai/v2/product_avatar/generation' \
  --header 'x-api-key: YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "product_image_url": "https://res.jogg.ai/product.jpg",
    "avatar_source": {
      "type": "id",
      "avatar_id": 1,
      "avatar_type": "public"
    },
    "quality": "high",
    "aspect_ratio": "3:2",
    "num_images": 4
  }'
```
